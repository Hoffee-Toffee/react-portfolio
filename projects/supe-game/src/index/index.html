What is Supe Game?

Supe Game is a game where you can create and battle with your own superpowered characters.
It's inspired by Pokemon battles, but with major differences; For example, there are no Pokemon.

Features

Character Customization

  • Custom Abilities and Moves
  • Custom Stats (Strengths and Weaknesses)
  • Custom Inventory (Items, Weapons, Armor)
  • Custom Appearance (Coming Soon)

Game Modes

  • Battles
    • CvC - Two players, each controlling one character, battle each other.
    • PvP - Two players, each controlling a team, battle each other.
    • TvT - Two teams, with each player controlling one character, battle each other.
  • Tournaments - (Coming soon)
    • CvC Tournament - CvC battles are held in a tournament format.
    • PvP Tournament - PvP battles are held in a tournament format.
    • TvT Tournament - TvT battles are held in a tournament format.
    • Mixed Tournament - PvP and TvT battles are held in a tournament format.
